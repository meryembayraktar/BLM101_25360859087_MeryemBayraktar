# Proje Konusu: Veri Manipülasyonu Ve Mantık Kapıları

# Proje Açıklaması:
Bu projede Veri Manipülasyonu Ve Mantık Kapıları konusu anlatılmak üzere  pdf formatında slayt, phyton kodu ve youtube konu anlatım videosundan oluşmaktadır.
Yazdığım kod temel matık kapılarını(AND,OR XOR,NOT) phyton programla dili kullanılaral simüle eden konsol tabanlı bir uygulamadır. Proje, kullanıcıdan alınan girişlere göre mantık kapılarında oluşan sonuçları öğretmeyi amaçlamaktadır.
Kodda herhangi bir extra kütüphane kullanılmamıştır. 
Kodun çalışması için bilgisayarınızda phyton 3 kurulu olması yeterlidir,harici bir kurulum gerektirmez.

# Algoritma Mantığı:

1. Program başlar
2. Kullanıcıdan A ve B girişleri alınır
3. Girişlerin sadece 0 veya 1 olduğu kontrol edilir
4. Kullanıcıdan mantık kapısı seçmesi istenir
5. Seçilen kapıya göre sonuç hesaplanır
6. Sonuç ekrana yazdırılır
7. 3 farklı mantık ifadesi için doğruluk tabloları oluşturulur
8. Program sonlanır
